# 通用图像生成编辑器详细参考

本文档作为 `image-generation` 技能的补充参考，包含详细的分辨率对照表、提示词工程指南与故障排查。

## 配置

所有 API Key 统一在 `~/.hermes/.env` 中配置：

```bash
BAILIAN_API_KEY=sk-xxxxxxxxxxxxxxxx
ZHIPU_API_KEY=your_zhipu_api_key
```

默认输出目录：`~/.hermes/images/`

## 详细分辨率表

### WAN 2.7

| 分辨率 | 比例 | 适用场景 |
|--------|------|----------|
| 1280*1280 | 1:1 | 通用、头像（默认） |
| 720*1280 | 9:16 | 自拍、全身照、手机壁纸（推荐） |
| 853*1280 | 2:3 | 半身照、竖版构图 |
| 1024*1536 | 2:3 | 竖版人像 |
| 1080*1920 | 9:16 | 全高清竖版、手机壁纸 |
| 1280*720 | 16:9 | 横版场景、电脑壁纸 |
| 1280*853 | 3:2 | 横版人像 |
| 1536*1024 | 3:2 | 横版场景 |
| 1920*1080 | 16:9 | 全高清横版、电脑壁纸 |
| 2048*2048 | 1:1 | 超高清正方形 |

### CogView

- 1024x1024（正方形，默认）

## 故障排查

### 问题：API Key 错误

**解决方案**：
1. 检查 `~/.hermes/.env` 中的 API Key 是否正确
2. 确认环境变量已正确配置
3. 检查 API 额度是否充足

### 问题：参考图加载失败

**解决方案**：
1. 检查文件路径是否正确
2. 确认图片文件存在且可读
3. 检查图片格式是否支持（JPG/PNG/WebP）

### 问题：生成速度慢

**解决方案**：
1. 避开使用高峰期
2. 减小图片分辨率
3. 组图模式支持异步（脚本已实现轮询等待）

## 注意事项

1. **API Key 安全**：不要将 `.env` 文件提交到代码仓库
2. **图生图限制**：只有 wan2.7-image 支持，cogview 不支持
3. **自动切换**：图生图时会自动切换到 wan
4. **分辨率格式**：wan 用 `*` 分隔（如 `1280*1280`），cogview 用 `x` 分隔（如 `1024x1024`）
5. **免费模型**：cogview-3-flash 免费，请合理使用
